#  Sistema de Cadastro Simples — Python

Este é um projeto básico em Python para treinar fundamentos importantes da programação, como:

- Funções
- Listas
- Estruturas condicionais
- Laços de repetição
- Tratamento de erros (`try/except`)
- Limpeza de entradas com `.strip()`
- Organização de código com `main()`

---

## Funcionalidades

✔ Cadastrar nomes  
✔ Listar nomes cadastrados  
✔ Remover nomes da lista  
✔ Menu interativo  
✔ Tratamento de erros para evitar que o programa quebre  
✔ Código simples, direto e fácil de entender  

---

##  Estrutura

O sistema funciona através de quatro funções principais:

- `cadastrar()` — adiciona nomes à lista  
- `listar()` — mostra todos os nomes cadastrados  
- `remover_usuario()` — remove um nome escolhido  
- `main()` — controla o menu do programa  

---

## Como executar

1. Instale o Python (3.x)
2. Baixe o arquivo `.py`
3. Abra o terminal na pasta onde o arquivo está
4. Execute:

python nome_do_arquivo.py



##  Objetivo do projeto

Este projeto foi feito com finalidade de estudo.  
O objetivo principal é treinar lógica de programação e organização de código em Python criando um sistema simples de CRUD (Criar, Ler, Apagar).
